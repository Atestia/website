Submit your implementation

If you have built a Tier-4-conformant implementation, submit it to the registry by emailing registry@tier4.org with:

  • Implementation name and version
  • Source URL (GitHub, GitLab, or equivalent)
  • License declaration
  • Spec version it conforms to (v1.0 or v1.1)
  • Declared conformance level (Implementer / Conformant / Certified)
  • If claiming Conformant level: link to passing Conformance Test Suite results
  • If claiming Certified level: link to current Certificate of Tier-4 v1.1 Certification

Listings are maintained by the Tier-4 Compliance Working Group. Misrepresentations of conformance level may result in removal.
