Architecture

Hosting: AWS US-East-1 (Virginia) primary. Single-tenant data plane in customer-controlled VPC for Enterprise tier. Shared multi-tenant data plane for Atestia Pro Starter/Pro/Scale subscriptions.

Compute: ECS Fargate (containerized). Auto-scaling groups. Multi-AZ for high availability.

Storage: RDS PostgreSQL Multi-AZ for hot lineage data (90 days). S3 (S3-IA tier) with KMS envelope encryption for cold lineage data (90 days+, up to 7 years per SR 26-2). Customer-held CMK supported for Enterprise.

Network: AWS WAF + CloudFront edge. VPC isolation between tenants in shared tier. Private VPC for Enterprise tier. No public ingress to data plane.

Identity: Clerk (consumer/dev tier) + SAML/SCIM SSO for Enterprise. MFA required for admin. JIT access for support.

Secrets: AWS Secrets Manager + HashiCorp Vault (HCP). Rotation every 90 days.
