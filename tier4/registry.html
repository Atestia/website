Tier-4 Conformant Implementations

A public list of implementations that have declared conformance with Tier-4 v1.1 at the Implementer, Conformant, or Certified level. Listing is free for any party.

Implementer level

Implementations that emit Tier-4-conformant Lineage records per §4 of the spec.

Conformant level

Implementations that pass the Tier-4 Conformance Test Suite covering §4, §5, §6, §7, and §8 of the spec.

No implementations have yet completed the Conformance Test Suite. Implementations expected: Q4 2026.

Certified level

Implementations that have undergone an annual third-party audit by a Working Group-approved auditor.

No implementations have yet been certified. Certification cycle begins Q1 2027.

Submit your implementation

If you have built a Tier-4-conformant implementation, submit it to the registry by emailing registry@tier4.org with:

  • · Implementation name and version
  • · Source URL (GitHub, GitLab, or equivalent)
  • · License declaration
  • · Spec version it conforms to (v1.0 or v1.1)
  • · Declared conformance level (Implementer / Conformant / Certified)
  • · If claiming Conformant level: link to passing Conformance Test Suite results
  • · If claiming Certified level: link to current Certificate of Tier-4 v1.1 Certification
