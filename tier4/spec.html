Working Draft v1.1

Tier-4 Open Reference Architecture for SR 26-2 Vendor Model Attestation

Status: v1.1 working draft. Public RFC period: 2026-06-01 → 2026-07-31. Ratification target: 2026-09-30.

Read the spec

The Tier-4 v1.1 specification is 4,200 words across 13 sections and 2 appendices. It defines the canonical schema, the OCC 2013-29 packet format, the SR 26-2 attestation pack, the BCBS 239 aggregation reference, the retention model, conformance levels, and cross-spec mapping to FFIEC and NIST AI RMF.

RFC process

v1.1 enters 60-day public comment period on 2026-06-01. Anyone may file an RFC at github.com/Atestia/tier4-mcp-server/issues proposing additions, clarifications, or modifications. The Working Group sub-committee responds to comments and produces a revised draft. The Steering Committee votes on adoption.

v1.1 ratification target: 2026-09-30.

Conformance Test Suite

The Conformance Test Suite at github.com/Atestia/tier4-mcp-server/tree/main/examples includes validation cases for every schema element, packet generator, and conformance level. Implementations passing the test suite may use the descriptor "Tier-4 v1.1 Conformant".
