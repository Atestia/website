The universal evidence layer for regulated AI

The runtime evidence layer for AI-powered banks.

Atestia captures every vendor model invocation, every risk data lineage event, and every agent reasoning chain at runtime, in production, examiner-ready. Banks today. Capital markets, insurance, healthcare next.

See the product Open-source on GitHub

Built on Anthropic MCP. OpenLineage compatible. MPL-2.0. Governed by the Tier-4 Compliance Working Group (501(c)(6)).

Product

One MCP server. Ten tools. Three regulations at launch.

Lineage capture

Every vendor model call

SHA-256 input hash + vendor ID + agent ID + latency + decision context, persisted at invocation time.

Reasoning capture

Every agent decision

Observation → tool call → tool result → decision, with confidence + human-in-loop status.

Packet generation

Examiner-ready on demand

OCC 2013-29, SR 26-2, BCBS 239, ECOA Reg B Adverse Action — generated in seconds.
